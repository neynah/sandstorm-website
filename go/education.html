Here's why the IT team will love it

Sandstorm comes with + great apps for every project. Instructors and students can also package their own apps that anyone can then install.

  • On-prem install for education privacy

    Run Sandstorm on hardware that you control. Everything users do on a Sandstorm server stays safely within your institution. Enjoy free dynamic DNS and SSL.

  • Self-service software

    Sandstorm provides simple web access to every user at the institution. When you approve Sandstorm, users do what they want safely within Sandstorm, and they file fewer support tickets.

  • Single-sign-on and web access

    Once you enable login via Google, GitHub, or email, every app automatically uses that login provider. LDAP and SAML are to arrive soon.

  • Custom apps can use the built-in access control

    Academic departments need custom apps. As those apps get older, Sandstorm helps you sleep well at night. Sandstorm automatically mitigates against 95% of app security issues, according to our analysis.

  • Access control like Google Docs

    Sandstorm makes all data private by default. Users can share access through a familiar interface.

  • Automatic app downscaling

    When a user closes a browser tab to an app, Sandstorm waits a few minutes, then terminates the processes. When the user visits the app again, Sandstorm starts it automatically.

  • Centralized backups and data migration

    Your Sandstorm server stores all data for all apps and all the databases they use. When you take a backup of it, you have everything.

  • Open source software

    Sandstorm is open source software enjoyed by thousands of individuals at non-profits and companies every day. The company behind Sandstorm is building Sandstorm for Work, a paid offering adding more SSO integration.
